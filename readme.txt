== Description ==
**Motive** This plugin is used to sync the email address of woocommerce orders to Autoresponders.
Whenever the plugin's cron hit, it fetches all the complete woocommerce orders email id and sync them to the connected Autoresponders account.

New Woocommerce Orders are used to sync when cron runs.Also if any order is updated then that Order will also be used to sync on autoresponders.

**Free Version** 
Sync WooCommerce Orders Email of Customers to connected account of :

1. Mailchimp
2. Infusionsoft
3. Imnicamail


**Premium Version** 
Sync WooCommerce Orders Email of Customers to connected account of :

1. Mailchimp
2. Infusionsoft
3. GetResponse
4. Imnicamail
5. iContact
6. Sendreach
7. Aweber
8. Klick-Tipp


APIs of separate Autoresponders are used to sync emails.

== Installation ==

For a manual installation via FTP:

1. Extract the downloaded plugin and upload to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' screen in your WordPress admin area
3. You will see a menu at the left hand side named with "WOAR"
4. Enter Email id and License Key in License Tab (skip this if using Free version)
5. Enter Corresponding Autoresponder's detail to connect in Autoresponders Tab
6. Either Check WP cron to use Wordpress cron or you can set cpanel cron also.

To upload the plugin through WordPress, instead of FTP:

1. Upload the downloaded zip file on the 'Add New' plugins screen (see the 'Upload' tab) in your WordPress admin area and activate.
2. Activate the plugin through the 'Plugins' screen in your WordPress admin area
3. You will see a menu at the left hand side named with "WOAR"
4. Enter Email id and License Key in License Tab (skip this if using Free version)
5. Enter Corresponding Autoresponder's detail to connect in Autoresponders Tab
6. Either Check WP cron to use Wordpress cron or you can set cpanel cron also.
